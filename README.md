**Spec-Forge API** is an engine designed to transform raw data sources, including **JSON examples**, **JSON schemas**, and **strictly formated HTML tables** into a unified **Intermediate Representation (IR)**. This IR is then used to generate clean, production-ready **Java** and **TypeScript** codebases.
